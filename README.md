# АЗС Онлайн

Мобильное PWA-приложение для учёта очереди, предварительной записи и контроля отпуска топлива на АЗС.

## Назначение

Приложение помогает сотрудникам АЗС и администрации контролировать:

- запись автомобилей на дату;
- очередь на текущий день;
- проверку допуска по госномеру;
- запрет повторной заправки;
- лимиты по АЗС и видам топлива;
- фиксацию факта заправки;
- работу при плохом интернете;
- синхронизацию офлайн-операций;
- отчёты и журнал действий.

## Главные правила

1. Главный идентификатор автомобиля — нормализованный госномер.
2. Приложение работает с 3 АЗС.
3. Один обычный автомобиль может заправиться только один раз в календарный день по всем 3 АЗС.
4. Запись доступна на любую дату, по умолчанию открывается завтрашняя дата.
5. Если записи на текущую дату нет — заправка запрещена.
6. Ручное разрешение может дать только старший смены или администратор.
7. Лимиты создаёт админ или старший смены.
8. Оператор записывает автомобили.
9. Кассир проверяет и фиксирует факт заправки.
10. Офлайн-операции считаются неподтверждёнными до синхронизации с сервером.
